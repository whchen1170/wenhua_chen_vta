this is the VTA project

the deployment instruction:

1) Build your jar file using mvn.
cd to your source folder, and run "mvn clean package"

my example command:
cd ~/git/Projects-master/BearMaps
mvn clean package

2) This creates your fat jar file inside the target directory. Copy the jar file to any location on the server.
Go to the your bin folder of your spark. ( in my case: ~/app/spark-2.1.1-bin-hadoop2.7/bin)

3) submitting your application using spark-submit.
wchen@:~/app/spark-2.1.1-bin-hadoop2.7/bin$ ./spark-submit --class "MapServer" --master local /Users/wchen/git/Projects-master/BearMaps/target/map-1.0.jar
